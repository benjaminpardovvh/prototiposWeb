Introduccion al lenguaje

HTML es el esqueleto de una pagina web, este lenguaje permite:
-escribir texto usando distintas etiquetas, como "h1" para los titulos grandes, "p" para parrafos, etcetera
-Añadir videos con la etiqueta source seguido de la ubicacion del archivo a mostrar.
-Colocar imagenes con la etiqueta "img" seguido de la ruta de la imagen, tambien puedes poner un texto descriptivo en caso de no poder ver la imagen, esto se hace con el atributo "alt".
-Añadir texto con links que te redireccionan a otras paginas. y no solo eso, tambien puede ser enlazado con java script para añadirle funcionalidad a ciertas cosas como, por ejemplo, los botones de una pagina.
con "css" puedes estilizar la pagina, es un lenguaje distinto que HTML y java script, css se orienta mas a lo visual y java script es para dare dinamismo a la pagina
por lo que recomiendo aprender un poco de cada uno para no estar tan perdidos al momento de querer implementar estilo u añadir dinamismo a la pagina

Etiquetas de Formato

Para poder hacer lo que tu conoces como "enter" o pasar a la siguiente linea, para que la pagina lo entienda como tal se usa la etiqueta "br" la cual nos permite como programadores, hacer el salto a la siguiente linea.
Esas lineas que puedes ver separando ciertas secciones de esta pagina son producto de la etiqueta "hr" la cual crea una linea horizontal, se recomienda darle un espacio dedicado a la etiqueta solo por el orden dentro del codigo.
